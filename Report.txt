1. Introduction

This project is a console-based quiz application developed in the C programming language. The program is designed to test a user’s knowledge of Wave Optics, covering topics such as interference, diffraction, and polarization.

The quiz consists of 25 multiple-choice questions, each having four options. The user selects an answer, and the program evaluates the response, provides feedback, and calculates the final score.

2. Objectives
To develop a basic quiz system using C.
To strengthen concepts of structures and arrays.
To implement input validation in user interaction.
To evaluate user performance with scoring and feedback.
3. Features of the Program
25 pre-defined questions from Wave Optics.
Multiple-choice format (4 options per question).
Input validation for:
Non-numeric inputs

Final score and percentage calculation.
